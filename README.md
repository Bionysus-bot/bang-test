# bang-test

> 一个轻轻巧巧、可可爱爱的邦邦测试小项目。

这里是 `bang-test`，用来存放和整理邦邦相关的小测试。  
不管是角色、乐队、歌曲，还是一些有趣的小问题，都可以慢慢收进这里。

## 这是什么

`bang-test` 是一个围绕邦邦主题的小测试项目。  
它可以被用来做：

- 邦邦角色相关测试
- 乐队或歌曲偏好测试
- 小问卷、小测验、小互动
- 其他和邦邦有关的可爱点子

## 目标

这个项目希望做到：

- 简单好用
- 内容可爱
- 结构清楚
- 方便继续添加新的测试题

## 适合谁

- 喜欢 BanG Dream! 的朋友
- 想做一点邦邦相关小互动的人
- 想收集测试题、问卷或小游戏想法的人
- 只是路过但想看看可爱东西的人
